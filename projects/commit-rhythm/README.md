# 새벽의 커밋 — Commit Rhythm

3년치 커밋 기록으로 "나는 언제 코딩하는가"를 그린 데이터 스토리.
기여도 캘린더 · 펀치카드 · 월별 언어 스택 · 언어 도넛. **D3.js**(스케일·도형)와
손으로 짠 SVG.

**Live:** https://lamgul.github.io/projects/commit-rhythm/

## 정직하게

`data.json`은 실제 저장소 로그가 아니라, 시각화를 통제된 조건에서 시험하려고
**결정론적으로 생성한 샘플**이다(야행성·주중 편향·휴가 공백·언어 이동 패턴 포함).
씨앗이 고정이라 언제 봐도 같은 그림이 나온다.

중요한 건 데이터가 아니라 파이프라인이다. `viz.js`는 `{ daily, punch, monthly, langs }`
모양의 JSON만 받으면 되고, 진짜 `git log` 출력을 같은 모양으로 집계하면 실제 리듬이
그대로 그려진다.

## 메모

- 캘린더/펀치카드는 강조색 위에 `fill-opacity`만 얹어 **테마 전환에 자동 대응**한다
  (색을 절대값으로 박지 않는다).
- D3는 `scaleSqrt / scaleLinear / arc / pie`만. 레이아웃은 직접 계산 →
  좌표가 어디서 오는지 항상 안다.
