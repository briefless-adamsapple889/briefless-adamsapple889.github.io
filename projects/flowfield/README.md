# 흐름 — Flowfield

값 노이즈로 만든 벡터장 위를 수천 개의 입자가 흐르는 제너러티브 캔버스.
마우스로 밀어낼 수 있다. 순수 Canvas 2D, **의존성 0**.

**Live:** https://lamgul.github.io/projects/flowfield/

## 재미있었던 지점

- **값 노이즈를 직접.** 격자 해시 + 부드러운 보간으로 연속적인 각도장을 만든다.
  무작위가 아니라 '결'이 있어야 흐름이 된다.
- **배치 스트로크.** 입자를 각도로 색 버킷에 나눠, 같은 색 선분을 한 `Path`에 모아
  버킷당 한 번씩만 그린다 → 수천 개의 선이 네 번의 draw call로.
- **잔상은 안개로.** 화면을 지우지 않고 매 프레임 배경을 6% 투명도로 덮는다.
- **안 보이면 쉰다.** `IntersectionObserver`로 오프스크린이면 루프 정지,
  `prefers-reduced-motion`이면 정지 화면 한 장.

`flow.js` 한 파일이 전부다.
